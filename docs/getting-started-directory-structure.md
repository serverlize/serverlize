title: Directory Structure
---

Once inside the project, you should have a folder structure as follows:

```
├── app
│   └── Functions
│       └── HelloWorld.ts
├── jest.config.js
├── package.json
├── prettier.config.js
├── tsconfig.json
├── tsconfig.test.json
└── tslint.json
```
