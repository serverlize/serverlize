title: Models
---

The CLI tool helps you get the mundane jobs done quickly.

```bash
$ slz model create
$ slz model:create <name> --hashKey <hashKey> \
  [--rangeKey <rangeKey>] \
  [--capacity 5,5]
$ slz model:create <name> --hashKey <hashKey> \
  [--rangeKey <rangeKey>] \
  [--autoscale 5-500,70]
```
